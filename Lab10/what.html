What is the Web Animations API?

Paragraph 1. Web Animations API is a browser-standard interface that gives developers access to the browser’s own animation engine, allowing them to animate DOM elements using JavaScript. Rather than relying entirely on CSS keyframe animations or transitions, WAAPI defines objects like Key frame Effect, Animation, and a shared timeline to make sure that animations, timing, and playblack can all be managed with code.

Paragraph 2. .

Behind the scenes, Web Animation API works using two cooperating models being, the Timing Model which tracks when things should start, how long they last, playback rate, iterations, etc and the Animation Model which defines what the visual changes are the keyframes that transform element properties over time.

Paragraph 3. A simple way of using Web Animation API is via the element.animate() method that gives it a list of Keyframes, and an options object, and the browser will animate the element.
