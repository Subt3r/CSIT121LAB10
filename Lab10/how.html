How to Use the Web Animations API

What Why How (Examples) Sources

Examples and patterns

Paragraph 1. Using the Web Animation API usually starts by selecting an element and calling the animate method on it. this method takes two things, a set of keyframes and some animation settings. Keyframes describe change over a course of time, such as an element moving or fading.

Paragraph 2. These elements can use simple Keyframes to animate basic properties. An example is, you can fade elements from insivible to visible or even move them back and forth from one position to another. The animation settings even let you choose how long the animation will last and how many times if at all it repeats.

Paragraph 3. You can also interact with aniamtions after they start. An example would be if you were to pause, reverse, or change the speed of the animation using JavaScript. This allows developers to make their animations respond to buttons, scrolling, or any other actions, which can create more interactive web pages.

Live Demo

This demo shows a simple animation created and controlled via the Web Animations API. Use the accessible controls below to play, pause, or reverse.

Code (Web Animations API)


// create animation
const box = document.getElementById('animBox');
const animation = box.animate(
  [
    { transform: 'translateY(0) rotate(0deg)', opacity:1 },
    { transform: 'translateY(-40px) rotate(10deg)', opacity:0.9 },
    { transform: 'translateY(0) rotate(0deg)', opacity:1 }
  ],
  {
    duration: 1400,
    iterations: Infinity,
    easing: 'cubic-bezier(.36,.07,.19,.97)'
  }
);

// controls
document.getElementById('playBtn').addEventListener('click', ()=>animation.play());
document.getElementById('pauseBtn').addEventListener('click', ()=>animation.pause());
document.getElementById('reverseBtn').addEventListener('click', ()=>animation.reverse());
document.getElementById('finishBtn').addEventListener('click', ()=>animation.finish());

Example: Web Animations API used to create performant, script-driven animations.
